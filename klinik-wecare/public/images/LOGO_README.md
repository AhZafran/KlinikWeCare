# Logo Placement Guide

## Instructions
Place the clinic logo file in this directory with the following naming:

**Filename:** `logo.png` or `logo.svg`

**Recommended specifications:**
- Format: SVG (preferred) or PNG with transparent background
- Dimensions: 200-300px width (height will auto-scale)
- File size: < 100KB for optimal performance
